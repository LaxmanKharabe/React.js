REST Parameter

REST parameter allows a function to accept multiple arguments as an array.

It is denoted by three dots (...) followed by the parameter name.

The REST parameter must be the last parameter in the function's parameter list, as it collects all the remaining arguments.

Every function have one and only REST parameter.

Syntax:

        function restPara (para1, para2, ...restParameter){
            // Function body
        }

Example: 

        function Details(...products) { // ...products is a REST parameter
            var [id, name, price, stock] = products;  // Array Destructuring : we can assign values from an array to variables.
            document.write(`ID: ${id} <br> Name: ${name} <br> Price: ${price} <br>Stock: ${stock} <br>`)
        }
        Details(101, "iPhone 14", 59000, "In Stock");

        function add (...values){
            var result = 0;
            for (const allValues of values){
               result += allValues;
            }
            return result;
        }
        document.write(`Addition is: ${add(12,10,1)}`);
