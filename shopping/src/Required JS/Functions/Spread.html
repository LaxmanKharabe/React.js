Spread: (...)
  • The spread syntax (...) is often referred to as the "spread operator" in JavaScript.
  • It is widely used for spreading the elements of an iterable (like an array) or the properties of an object.
  • Spread is an approach of configuring the one actual value that spreading content into multiple formal parameters.
  • Spread value must be a collection[].
Syntax:
...iterable
Example: 
        EX1:
        const products = [101, "iPhone 13", 59000, "In Stock"];
        var [id, name, price, stock] = products;
        console.log(...products); // Spread


        //Ex2: Copying Objects:
        const student = { name: "John", age: 30 };
        const newStudent = {...student};
        console.log(student); // Output: { name: "John", age: 30 }
        console.log(newStudent); // Output: { name: "John", age: 30 }

        //Ex3: Combining Arrays
        const numbers1 = [1, 2, 3];
        const numbers2 = [4, 5, 6];
        const combinedNumbers = [...numbers1, ...numbers2];
        console.log(combinedNumbers); // Output: [1, 2, 3, 4, 5, 6]

        //Ex4: Adding element to an existing array:
        array1.push(...["pushed element1", "pushed element2"]);
        console.log(array1);

        //Ex5: Coping an array:
        const copiedArray1 = [...array1];
        console.log("Copied array: " + copiedArray1);

        //Ex6: Adding properties to an existing object:
        const newStudentObj = {...student, id: 101, course: "JavaScript"} // Added new properties while copying.
        console.log(newStudentObj);
