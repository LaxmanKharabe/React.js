Parameter:
A parameter is a variable that is used in the function declaration.
Purpose of Parameters:
Parameters act as placeholders for values that the function will receive when it is called.
Syntax: 
            function exampleFunction(parameter1, parameter2) {
                // Function body
            }
Argument:
An argument is a value that is passed to a function when calling or invoking that function.
Purpose of Argument:
Arguments supply specific data to the parameters defined in the function.
Syntax:
exampleFunction(argument1, argument2);
Example: 
        // Function declaration with parameters
        function add(number1, number2) {
            // Function body: Adds two numbers and returns the result
            return number1 + number2;
        }

        // Function call with arguments
        const result = add(5, 3);

        // Output the result
        console.log(result); // Output: 8
